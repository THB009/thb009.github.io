# 第五章 轩然大波

经过了十几秒的沉默之后，上条让医生过来，小声地在他耳边问了句：“那，我宿舍怎么办？万一有小偷进来呢？”而医生似乎是早有准备，话音刚落，他就紧接着回答道：“放心吧，警备员逮捕你的时候，早就已经把除私人物品以外的东西全部封存了。好了，时候真不早了，你们的选择是？”上条和茵蒂克丝齐声回答道：“行吧。”医生听后，微微地笑了笑，然后打了个响指。之后的一切，在这两人眼里如走马灯一般在眼前闪过。病房内突然闯出来几名警备员，那名医生则直接从病房走了出去，上条、茵蒂克丝在他们的簇拥下从楼梯口进到地下车库。而昏暗的车库里早已经有五辆装甲车等着了，他们坐上了最中间的装甲车，随后车队鱼贯而出，径直驶离了这家医院。

路程很顺，一次减速都没有，只是过了很久，头顶上逐渐传来了巨大的轰鸣声，搞得这两人有点不适。大概行驶了约有一个小时，车辆才渐渐减速，直至停了下来。坐在上条旁边的警备员叫他先别动，随后自己下了车。又过了几分钟，他走回来，说了句：“请出来吧。”

上条刚一下车，就看到一架双发大型客机，机身上涂着蓝红深蓝三个条纹，垂直尾翼上还有一个中国国旗，最末端的舱门上涂着解放军的标志。虽然比他之前和茵蒂克丝去意大利时坐的波音747-200和回程时的高超音速飞机略逊一筹，但一眼看过去，这飞机实在是如同碧日蓝天下亭亭玉立的少女一般，让他的目光时刻不能从它身上移开。直到来自头上的强烈气流吹入他的眼睛时，他才眯着眼，发现那轰鸣声的来源是不知什么时候盘旋在车队头上的两架直-19。

舱门外，登机梯显然已经等候多时，两侧站着几个解放军，他们穿着作训服，手上拿着的QBZ-191在正午的太阳下闪耀着淡淡的金属光泽，显得更为瞩目。望着这般场景，上条二人只得沿着中间的预留的位置机械地登上飞机。在登机梯上，上条下意识地往回看了一眼，远处的停机坪上整齐划一地停满了歼-31和歼-20战斗机。

刚一踏进飞机，几名西装革履的男子就从头等舱走了过来，其中为首带着墨镜的说：“您们两位好，我姓张，叫我小张就好，是这次行程的负责人，有什么事叫我就好。这趟航班只有机组、安保和您们，所以这架飞机的座位，你们可以随便选。不过，如果你需要下位活动的话，请一定要按下头顶上的呼叫铃，以便于我联系。如果没有什么其他问题的话，我就先行告辞了。”“那个，我想问问，这架飞机是？”上条谨慎地问道，“这架飞机是我们中国商飞自行研制的C929，具体型号是C929-700，外国人喜欢叫它CR929，是我国继C919之后的第三款自研民航客机，主要对标的是空客A350XWB和波音787。”“我知道了，多谢。”

![CR929 Series]( /images/cr929xilie.jpg "C929系列双发宽体远程客机，从前至后分别为C929-700型，C929-600型，C929-500型")

在说完这句恭维话之后，上条下意识地就往经济舱的位置走去，但刚刚跨出步子，茵蒂克丝大喊道：“当麻！”上条循声望去，发现茵蒂克丝早已经坐在中间的两个头等舱座位的一个上了。“哦，来了！”上条连忙走了过来，接着在茵蒂克丝旁边空着的座位上坐下。

![C929 First Class 1]( /images/C929-2.jpg "C929飞机的头等舱（1）")
![C929 First Class 2]( /images/C929-3.jpg "C929飞机的头等舱（2）")

“各位乘客，飞机即将开始滑行，请各位乘客迅速回到座位上，系好安全带，收起小桌板，感谢您的配合！”

此时，小张和那几个人也正从经济舱那边赶了过来，给他们大致讲了下这次飞行的信息。座位前嵌入的屏幕也开始播放惯例的安全须知。

这下子，上条终于有闲心来好好体验一下这架飞机的头等舱了，平生连飞机都很少坐的他，今天可以好好享受一次这样豪华的飞行了。“不愧是中国人，连这种头戴式耳机都能每个头等舱配一个啊！”上条戴着早就放在一旁的耳机，赞叹道。“当麻！当麻！这架飞机真的好漂亮啊！”虽然茵蒂克丝是英国人，但还是对眼前来自东方大国的这家飞机惊叹无比。

上条看着眼前这个嵌入的显示屏，不知什么时候语言已经切换成了日语，尺寸小是小了点，不过清晰度和像素方面可比整个学园都市的产品质量都不知好了多少，用手在上面滑动了好几下，几乎没有延迟，非常灵敏。“真不愧是中国产品啊！”他心里这样想到。

过了几分钟，飞机在跑道上完成了最后的起飞前检查，随后丝滑地加推力，微弱的引擎声从机翼两边传来，如果不是窗外的景象开始倾斜，这两人甚至都还以为飞机还在跑道上呢。直到上条望向窗外，景色已经倾斜了一定角度才意识到：“哦，起飞啦。”

从学园都市到北京只需约三四个小时的飞行时间，所以路程上自然是没有餐食之类的，这着实让茵蒂克丝这个小吃货有些不满，不过这长度的时间，说长不长，说短也不短。两人内心最主要的想法都是想要尽快到达中国，看看真正的中国到底是什么样子。

看着眼前这个显示屏上有个机外摄像头显示的功能，上条觉得有点意思，于是便就点开了这个应用，扑面而来的是四个摄像头的预览，分别是机腹摄像头、左机翼、右机翼摄像头和垂直尾翼上的摄像头。他随便选了一个左机翼摄像头，画质很清晰，色彩饱满。唯一一个显得突兀的是，画面上有三架歼-31在左翼进行伴随飞行。上条惊异，又随即切换成了右翼摄像头，一样有三架歼-31在跟随着飞行。这下他彻底清楚了，中国人为了保护他们的安全，在安保上竟然给他这种普通人配备了如此高规格的安保措施，实属罕见。

无聊之余，上条只得看点视频什么的打发下时间，跟他在网上看到的一样，所有的多媒体都是免费的，更新也很及时，但动漫什么的似乎有点少，于是他只能看着排行榜上推荐的朝鲜战争纪录片，学一些中国的历史知识。

影片中，1950年11月21日，中国人民志愿军一百辆“喀秋莎”多管火箭炮疯狂地朝平壤北城区展开了猛烈的炮击，火箭炮履带上的40枚132毫米，射程40公里的火箭呼啦啦地射向联合国军驻地，平壤市的北城区大部分都宛如地狱一般遭受了饱和的火箭弹攻击，无数的美军和南朝鲜伪军士兵在睡梦中就被炸的灰飞烟灭。

“原，原来中国这么早就能把美军打的屁滚尿流了吗？”上条的内心震撼不已，那可是六十几年前啊，当时的中国才刚刚建国，收复了台湾，紧接着就开始打朝鲜战争，结果还是硬生生把美军从鸭绿江边推回三八线，并且反攻到了江原南道的马山市[^1]，逼着美国签订了和谈协定。

[^1]:现实世界中，马山市已于2010年合并进昌原市。

61米宽的巨鹰在万米高空的蔚蓝天幕上，从地平线的一端到另一端，缓缓地拖出了两道尾迹云。而在一万三千米之下，一个睡眼朦胧的年轻人正懒散地躺在柔软的床上，望着全息投影将房屋正上方的景象投射的天花板，万里无云的天空只有稀稀疏疏的十二束白色细线从上方延伸到下方。

“这是警备员的还是中国的？看起来应该是在护航飞行吧。”想到这里，他扭了下自己的头，看了看床头柜上放着的照片，那是他在拉斯维加斯会展中心参加DEF CON夺得桂冠时留下的珍贵回忆。照片中的自己身着科研人员经典的白色大褂，胸前悬挂象征黑客至高荣誉的骷髅头黑色胸章，上面刻着为人所熟知的网名——C0deBreak3r，右手抱着一只木头鹰咧着嘴。照片背面则用蓝色马克笔写就“松村阪二先生 惠存”的日语字样。他仔细端详了很久，又将头转了回来，再次望着天花板，那十二根细线已经不知什么时候将天花板的对边给连接起来了。

窗外阳光洒进了用窗帘半掩着的卧室，松村阪二短暂地回顾了一下自己这二十六载的人生。

现在的他，是一名年纪轻轻却在计算机科学领域大放异彩的天才。仅仅十二岁，他就取得了国际信息学奥林匹克竞赛的金奖，十四岁获得国际大学生程序设计竞赛冠军，十六岁参加DEF CON CTF。虽然比起那位tourist大神[^2]来说要稍显逊色了些，但还是惊艳了比赛的评委老师。高中毕业后即前往清华大学信息科学技术学院进修。二十岁时，当别人还在为学位和工作焦虑的时候，他就已经是IEEE的会士之一，并在世界各大计算机会议上来回奔走。

[^2]: 指根纳季·科勒特克维奇（Геннадий Короткевич，1994年9月25日－），网名tourist，白俄罗斯计算机程序员，被誉为世界上最优秀的竞赛程序员之一。他在各类编程竞赛中屡获殊荣，包括11岁时创下国际信息学奥林匹克竞赛（IOI）选手最小年龄记录并六度获得 IOI 金牌、国际大学生程序设计竞赛（ICPC）第一名。科勒特克维奇在2014年至2021年期间六次获得Google编程挑战赛的冠军，并多次在Facebook黑客杯和Codeforces编程竞赛中取得第一名。他的编程技巧和解决问题的能力使他在全球编程社区中享有极高的声誉。

现在的他，是一位学园都市声名显赫的计算机网络安全专家。毕业以后，他被理事会以高薪聘为学园都市计算机网络工程的科研人员。经过四年科研领域的滚爬摸打，自己带领的科研小组成功设计出了以洋葱路由为基础进行大幅度改良的“明石”军民用网络，名声大震。他也从一个科研小组长升迁为了第三科研所网络安全和信息科长，速度之快，实属罕见。

按照这个趋势，除了感情方面外，未来应该是十分美好的。

至少在收到那条消息前，他是这么认为的。

松村仍是躺在床上，如一滩止水一般平静，又像一滩死水般静止不动。他双眼无光地打开了手机，反复看着那张图片——一张病危通知书。通知书的左上角，患者一栏用正楷体毫无感情地显示着自己父亲的名字。下面用一个表格详细地列出自己看不懂的各个指标，最终在下面的结果一栏中看到了一个很长一大串的病名。

松村搜了一下，这是一种很罕见的病症，只有一种治疗方法，而且全套有效护理加上后期康复治疗的费用，即使是自己这种级别的科研人员也是很难拿得出手的。更令他雪上加霜的是，父亲的病检查出来已经是濒临晚期了，虽然还是能救得回来，但是费用又要比之前高出一大截。就算是把自己这套房子抵押出去，换得的贷款还是不能够拯救父亲的生命。

松村放下了手机，在床上摆出了个“大”字形，天花板上的尾迹云已经向两侧扩散，像十几束墨水直直地射入水中，随后缓缓地散开。想着那张照片里自己抱着的那具木雕鹰，他自言自语道：“折翼之鹰，还能飞起来吗？”

......

在经历了近四小时风平浪静的飞行后，飞机开始平缓地下降。广播中，已经略显疲惫的小张尽量用着标准的日语让这两个人系好安全带，准备降落，然后就立即结束了广播。

飞机略微向下倾斜了下角度，开始下降。不知是因为有意为之，亦或是偶然，这次飞机的飞行路线竟然恰好能够从左侧看见整个北京城的全貌，在渐已西落的太阳光芒下，北京城外鳞次栉比的高楼大厦与城内历史悠久的古色建筑交相辉映，相得益彰，让人不禁有种时空错乱的感觉。

“哦~这就是北京吧，原来从上空看起来这么美，不愧是中国的首都啊！”上条通过飞机上固定在机翼上的摄像头，浏览到了这一被绝大多数刚到北京的游客惊叹不已的一幕。茵蒂克丝虽然没说什么，但是她脸上的神情却完完全全地表现出了震惊和赞叹。在充分观赏了这美景之后，一阵铃声响起，所有的遮光板一起降下，彻底挡住了外面的视线，上条的大脑这回倒是转的挺快的，立马就猜到了——这是要降落军用机场啊！

随着机头轻轻地向上稍微扬起些许，这架C929丝滑地降落在空军某机场的跑道上。渐渐地，在飞机速度降至正常滑行的30节时，小张又一次从经济舱走了过来，说道：“待会飞机停下来，你们出机场后，我们的人会给你们必要的文件，酒店什么的我们都已经安排好了，明天你们就可以自由活动。好了，祝你们在中国玩得愉快！”说完，他径直走到前面的舱门处，拉上了帘子，不知做什么去了。
过了几分钟，飞机彻底地停住了，安全带指示灯也熄灭了，两人解开了安全带，站了起来。小张也拉开帘子，对着三人提醒道：“你们的个人物品都已经装进飞机了，等会你们到酒店的时候，我们会给你们一起送过来的。那么，请到前面来吧！”

与来时几乎一样，打开的舱门前还是早已准备好的登机梯，旁边依旧是全副武装的解放军，只不过头上没有了直升机。上条特意最后一个起身，在即将跨出飞机舱门的瞬间，他悄悄问了下小张：“能不能让我看看这架飞机的驾驶舱？我对这飞机真的很感兴趣！”小张先是笑了笑，然后斩钉截铁地回答：“我得先询问上级，虽然是民航客机，不会有什么泄密问题。但因为政策原因，还是要申请一下的。”说完，他走进卫生间，关上了门。过了一会，他又出来说：“上面已经同意了，我这就让机组开门。”

“无与伦比。”这是上条走进C929驾驶舱的第一感受。虽然在民航爱好者看来，这飞机的驾驶舱像是空客A350和波音787的结合体。但对于出入境管理极其严格的学园都市而言，每月民航飞机的起降数量真可谓屈指可数。因此对于这种驾驶舱，他简直惊奇到下巴都要掉下来了。
![C929 Cookpit]( /images/C929-4.jpg "C929飞机的驾驶舱")

看着上条这样，小张发自内心地笑了，拍了拍他的肩膀，说：“上条先生，你明天也有时间来参观，赶紧跟着你那个小女孩走吧，不然找都找不到了！”上条似乎也意识到了自己的失态，挠了挠头，腼腆地笑了笑，然后迅速从驾驶舱离开，走下登机梯，看着已经在车边等候多时的军人们，他鞠了个躬，随后钻进了车里。随后，这辆红旗轿车便如出弓的箭矢一般离去了。
